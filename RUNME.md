# Running the Application

## Prerequisites
- [.NET 8 SDK](https://dotnet.microsoft.com/download)
- [Node.js 18+](https://nodejs.org) and npm

## Install Frontend Dependencies
cd FullStackTest/FullStackTest/ClientApp
npm install

## Run the Application
cd FullStackTest/FullStackTest
dotnet run

## Run Frontend Tests
cd FullStackTest/FullStackTest/ClientApp
npm test

## Run Frontend Linting
cd FullStackTest/FullStackTest/ClientApp
npm run lint
